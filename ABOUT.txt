# GitHub About 文案

## 中文 (Chinese)

**简短描述 (Short description):**
像素级精准的浏览器窗口尺寸调整工具，一键统一截图规范。

**详细描述 (Description):**
Reso-resizer（レゾリサイズ）是一款专为强迫症产品经理和开发者打造的 Chrome 扩展。它可以一键将浏览器窗口调整到精确像素尺寸，支持预置设备分辨率（iPhone、iPad、MacBook 等）和自定义输入，更具备智能 Viewport 模式，自动补偿浏览器工具栏占用空间，让网页可视区域精准匹配目标分辨率。

## English

**Short description:**
Pixel-perfect browser window resizing for standardized screenshots.

**Description:**
Reso-resizer (レゾリサイズ) is a Chrome extension crafted for perfectionist product managers and developers. Resize your browser window to exact pixel dimensions with one click. Features preset device resolutions (iPhone, iPad, MacBook, etc.), custom input, and an intelligent Viewport mode that compensates for browser UI elements to ensure the visible content area matches your target resolution precisely.
