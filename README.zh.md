# Reso-resizer / レゾリサイズ

还在为截图尺寸不统一而发愁吗？强迫症产品经理和开发者的福音来了！

Reso-resizer（レゾリサイズ）是一个极简优雅的 Chrome 浏览器插件，专为追求像素级精准的你而生。它能让你一键将浏览器窗口调整到任意精确分辨率，无论是为了统一截图规范，还是为了模拟不同设备的显示效果，都能轻松搞定。✨

🌟 核心魔法 (Features)

🎯 像素级精准控制：告别肉眼估算！输入目标分辨率，浏览器窗口瞬间调整到位，截图尺寸分毫不差。
🔄 预置/自定义双模式：

预置模式：一键选择常用分辨率（MacBook、iPhone、iPad 等），开箱即用。
自定义模式：自由输入任意宽高，满足你的特殊需求。


🖼️ 智能 Viewport 模式：可选仅调整网页可视区域（Viewport）大小，自动排除浏览器工具栏、地址栏和书签栏的高度，让网页内容区域精准匹配目标尺寸。
🌍 原生多语言支持：自动识别浏览器语言环境，中文、日语、英文无缝切换，无需手动设置。
⚙️ 配置文件驱动：所有预置分辨率集中管理在 config.json 中，修改配置无需改动代码，维护超轻松。


🧩 它是怎么工作的？
Reso-resizer（レゾリサイズ）的核心原理非常简单但高效：

标准模式：直接调用 Chrome API 调整整个浏览器窗口到指定尺寸。
Viewport 模式：

通过脚本注入获取当前网页的实际可视区域大小（innerWidth/innerHeight）。
计算浏览器边框和工具栏占用的额外空间。
自动补偿这些空间，最终让网页内容区域完美匹配你的目标分辨率。




🛠️ 安装与使用

下载或克隆本仓库到本地。
打开 Chrome 浏览器，进入 chrome://extensions/。
开启右上角的"开发者模式"。
点击"加载已解压的扩展程序"，选择本项目文件夹。
点击浏览器工具栏的 Reso-resizer 图标，开始精准控制！


⚙️ 如何自定义配置？
所有配置都集中在 config.json 文件中：

添加/修改预置分辨率：在 presets 数组中增减选项，支持多语言标签。
修改默认分辨率：调整 defaultResolution 值。
默认开启 Viewport 模式：将 defaultViewportOnly 设为 true。

配置文件自带详细的多语言注释，一看就懂！
